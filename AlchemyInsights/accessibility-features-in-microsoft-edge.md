# <a name="accessibility-features-in-microsoft-edge"></a>"Microsoft Edge" pritaikymo neįgaliesiems funkcijos

"Microsoft Edge" turi įtaisytąsias pritaikymo neįgaliesiems parinktis, padėsiančią naršyti žiniatinklį taip, kaip tai geriausiai atitinka jūsų poreikius.

Norėdami sužinoti daugiau, peržiūrėkite [pritaikymo neįgaliesiems funkcijas programoje "Microsoft Edge"](https://go.microsoft.com/fwlink/?linkid=2153648).
