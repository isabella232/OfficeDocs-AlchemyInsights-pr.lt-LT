# <a name="restore-a-deleted-sharepoint-subsite"></a>Panaikintų antrinių SharePoint atkūrimas

Panaikintos antrinės svetainės siunčiamos į svetainių rinkinio šiukšlinę, kur jos laikomos 93 dienas. Norėdami atkurti panaikintą anttrinę svetainės:
  
1. Naujame administravimo SharePoint raskite svetainę, iš kurios buvo panaikinta antrinė svetainė, ir įsitikinkite, kad esate svetainių rinkinio administratorius. 
    
2. Eikite į svetainę. Kairiojoje **srityje spustelėkite** Šiukšlinė. (Jei nematote šiukšlinės, spustelėkite piktogramą Parametrai, tada spustelėkite **Svetainės turinys**. Šiukšlinė yra viršutiniame dešiniajame komandų juostos kampe.)
    
3. Puslapio apačioje spustelėkite Antrojo **etapo šiukšlinė**.
    
4. Spustelėkite į kairę nuo antrinės svetainės, tada spustelėkite **Atkurti**.
